Progrés de la tercera setmana

Aquesta setmana he aconseguit entrar al thunderbird amb un compte mail del meu poste:

-Primer vaig haber de reiniciar la màquina completament i vaig a tornar a instalar el mkcert, el docker i el hugo.

-Després vaig crear la màquina amb l’ordre:

docker volume create volum_mail_server

docker run -p 443:443 -p 25:25 -p 110:110 -p 143:143 -p 465:465 -p 587:587 -p 993:993 -p 995:995 -p 4190:4190  -e TZ=Europe/Andorra -v volum_mail_server:/data --name "pastor.edu" -h "pastor.edu" -t analogic/poste.io

  • Aquesta ordre ens permet crear un contenidor docker amb tots els ports oberts (que s’utilitzin per aplicacións mail.)

  • Al acabar el pas anterior, vaig configurar el server i instalar el certificats.

Per finalitzar, vaig instalar thunderbird a clase i vaig configurarlo:

-Per configurar-lo vaig entrar al thunderbird desde base i vaig iniciar sessió amb el meu usuari del poste server (posant el seu nom, mail i contrasenya).

  • Després vaig clicar al botó de configuració de server i en el camp “server” vaig posar la ip de la màquina que conté el server (en aquest cas la de dmze) i vaig clicar al botó re-test (fa la comprobació de que hi hagi conexió).

  • Al acabar vaig clicar a “done”. Al clicar al botó va surtir una pestanya que deia que el certificat del server era insegur i que si volia fer una excepció (clarament que la vaig fer). Va entrar directament a la safata del usuari escollit.
