Progrés de la primera setmana

Aquesta setmana, el meu grup i jo ens hem dedicat en crear un container amb el docker per fer de servidor poste i configurar els domini a dintre del mateix de forma gràfica:

-Primer vam instalar docker utilitzant els seu propi repositori de apt.

-Després vam crear el container amb el docker run:

docker run
-p 443:443
-e TZ=Europe/Andorra
-v volum_mail_server:/data
–name pastor.edu
-h pastor.edu
-t analogic/poste.io

-Quan es va acabar de crear el container ens vam conectar al mateix desde el firefox de dmzf i vam configurar el domini amb les dades següents:

–Domini: jos-surfacad.edu –Estructura per els correus: “cognom”@jos-surfacad.edu

-Aquesta setmana també hem fet les següents comprovacions:

–Que un mail enviat desde un client del poste.io arriba als clients de google pero fas una resposta al mail desde google a la adreça del domini poste.io no apareix en el mail del poste.io –Que es poden tindre més d’un admin en un domini

Aquesta setmana no he avançat tant com esperaba degut a problemes amb la màquina virtual (La he hagut de tornar a fer) i que el propi blog ho he refet varies vegades per problemes amb la versió de hugo.
